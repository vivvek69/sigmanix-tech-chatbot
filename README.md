# 🎓 Sigmanix Tech AI Chatbot

A production-ready AI-powered educational chatbot built with Flask, React, LangChain, and Groq LLM.

## 🚀 Quick Deploy

### Deploy to Replit (Free, 24/7 Hosting)
1. Click: https://replit.com/new
2. Select Python
3. Upload files OR import from GitHub
4. Add GROQ_API_KEY in Secrets
5. Click Run

### Key Features
- 🤖 AI-powered with Groq LLM
- 🔍 RAG-based knowledge retrieval
- 📊 Conversation database
- ⭐ User feedback system
- 🎨 Beautiful UI
- 📱 Mobile responsive

## 📚 Features
- Answers about courses, duration, placements, registration
- Free-form Q&A with AI
- Quiz functionality
- Rating system

## 🔧 Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Set environment variables
export GROQ_API_KEY=your_key

# Run
python chatbot_production.py
```
